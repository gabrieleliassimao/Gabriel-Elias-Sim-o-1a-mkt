🌾 AgroTech 2077 - Projeto Agrinho
Este projeto é um site educativo de página única (Single Page Application) desenvolvido para o concurso Agrinho, abordando o tema "Tecnologia no Agronegócio e MST". O site combina um design futurista em tons de azul com conteúdo rico sobre sustentabilidade, reforma agrária e a revolução tecnológica no campo.

🚀 Sobre o Projeto
O objetivo do site é mostrar que a tecnologia não é inimiga do campo, mas sim uma ferramenta de transformação social e ambiental. O conteúdo destaca o papel do MST (Movimento dos Trabalhadores Rurais Sem Terra) na adoção da agroecologia tecnológica, agricultura de precisão e biofábricas, defendendo a soberania alimentar.

O design utiliza um estilo Cyberpunk/Futurista, com efeitos de "vidro" (glassmorphism), neon e animações em Canvas para transmitir modernidade.

✨ Funcionalidades
Design Responsivo: Adapta-se perfeitamente a celulares, tablets e computadores.
Animações de Scroll: Elementos aparecem suavemente conforme o usuário desce a página.
Background Interativo: Rede de partículas conectadas desenhada em HTML5 Canvas no topo da página.
Simulador de Irrigação: Ferramenta interativa em JavaScript que calcula a economia de água com base na tecnologia usada.
FAQ Interativo: Perguntas frequentes com funcionalidade de abrir/fechar (accordion).
Linha do Tempo: Visualização histórica da evolução da agricultura até a Agricultura 4.0.

💻 Como Executar
Para rodar este projeto, você não precisa instalar nenhum servidor complexo (como Node.js). Basta ter um navegador de internet.

Crie uma pasta no seu computador chamada projeto-agrinho.
Dentro dela, crie três arquivos:
index.html
style.css
script.js
Cole o código de cada linguagem nos arquivos correspondentes (fornecido nas etapas anteriores).
Dê um duplo clique no arquivo index.html para abrir no seu navegador (Chrome, Edge, Firefox, etc.).
Nota: Para ver as fontes personalizadas (Orbitron e Rajdhani) e as imagens de exemplo, o computador precisa estar conectado à internet.
